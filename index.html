About me

My name is Sean Hallahan. I’m a technology consultant who is moonlighting as a full stack web developer. I’m new to the development game but love learning and playing with new technologies. This website is where I will be posting anything useful I come across while flopping through my my first year as a developer. Hopefully I can help people through the some of the stumbling blocks I encounter and introduce you to some cool products/technologies.

In the daytime I work primarily with Java web applications using the Spring framework. In the evening I’m exploring serverless webapps, ReactJS, Node.js, and WordPress (for this blog). Really just trying to find my niche. I recently received my AWS Solutions Architect – Associate Certification, which means you’ll be seeing a lot of AWS architecture from me.
